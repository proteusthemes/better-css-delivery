== Description ==

To get the list of all the handles, visit your front WordPress page, add the `?debugBCD=true` to the end of the URL (ie. https://example.com/?debugBCD=true), view the source (Win/Linux: ctrl + U, Mac: cmd + alt + U) and scroll to the bottom. You should see the list of all registered CSS handles with respective URLs. The handles marked with `x` are enqueued and included in the current page load.

== Installation ==

1. Upload `better-css-delivery.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
